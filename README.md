Weather Data Pipeline with Python and PostgreSQL

Project Overview

This project builds a simple data pipeline that collects weather data from a public API, processes it, and stores it in a PostgreSQL database. The pipeline is containerized using Docker to ensure portability and easy deployment across different environments.

The goal of this project is to demonstrate core Data Engineering concepts, including:

Data extraction from an API
Data cleaning and transformation
Database schema design
ETL pipeline development
Containerization using Docker
Automation and logging
